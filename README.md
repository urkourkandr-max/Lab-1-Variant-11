# Lab-1-Variant-11 0.1.0
У цій лабораторній роботі реалізовано веб-застосунок для керування чергуваннями у лабораторії. Користувач може додавати нові чергування, редагувати існуючі записи, переглядати їх у таблиці, здійснювати пошук та сортування, а також видаляти записи. Реалізовано клієнтську валідацію даних з виведенням повідомлень про помилки. Дані зберігаються у масиві JavaScript та дублюються у localStorage, що дозволяє зберігати їх після перезавантаження сторінки. Відображення відбувається без перезавантаження сторінки.

1 Як запустити проєкт?
Завантажити або розпакувати папку проєкту.
Відкрити папку
Переконатися, що файли index.html, styles.css, app.js знаходяться в одній папці.
Двічі клікнути на файл index.html.
Проєкт відкриється у браузері.


2 Перевірка валідації
Натиснути кнопку "Зберегти" з порожніми полями ---- Повинні з’явитися повідомлення про помилки у відповідних незаповнених полях, а поля підсвітитися червоним.

Обрати дату в минулому ---- Повинна з’явитися помилка про некоректну дату

Не обрати час ---- Повинна з’явитися помилка з вимогою вибору часу

Ввести ім’я з цифрами ---- Повинна з’явитися помилка що дозволені лише літери та пробіли

Ввести ім’я з одним або двома символами ---- Повинна з’явитися помилка про мінімальну довжину імені не менше 3 символів

Ввести ім’я більше 30 символів ---- Повинна з’явитися помилка про перевищення максимальної довжини імені

Ввести коментар більше 50 символів ---- Повинне з’явитися повідомлення про обмеження довжини коментаря


3 Перевірка відображення даних
Після успішного додавання запису у таблиці повинні відображатися: порядковий номер, дата, час, ім’я студента та коментар

Кнопка "Редагувати"
Змінює данні запису у таблиці не змінюючи ID

Кнопка "Видалити"
Видаляє запис з таблиці

4 Перевірка додаткового функціоналу


Пошук:
Ввести текст у поле пошуку ---- таблиця повинна фільтрувати записи за іменем студента

Сортування:
Натиснути кнопку сортування ---- записи повинні відсортуватися за датою

Збереження даних:
Перезавантажити сторінку ---- додані записи повинні залишитися (завдяки localStorage).
